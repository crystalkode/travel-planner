# Travel Planner Frontend

Frontend application for the Travel Planner project.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm run build
```

## Status

This project is currently in early development.

Planned features include:

- Trip management
- Day-by-day itinerary planning
- Activity management
- Integration with the Travel Planner backend
